# Task 12.5: Remove Global atomRegistry

| Meta | Value |
|------|-------|
| **Task** | Remove global `atomRegistry` singleton and `createIsolatedRegistry()` |
| **Priority** | P2 — cleanup after migration |
| **Estimated effort** | Small |
| **Breaking changes** | YES — deprecate then remove |
| **Dependencies** | Task 12.2, Task 12.3, Task 12.4 |

## Context

After Tasks 12.1-12.4, every Store is self-contained with `ScopedRegistry`. The global `atomRegistry` singleton is no longer needed.

This task removes it in 2 phases to avoid breaking existing users.

**Files to modify:**
- `packages/core/src/atom-registry.ts` — deprecate then remove
- `packages/core/src/atom.ts` — remove unused import
- `packages/core/src/index.ts` — update exports
- `packages/core/src/store.ts` — update createStore signature
- All test files using `atomRegistry`

## Phase 1: Deprecation (this task)

### 1. Add deprecation warnings to atom-registry.ts

```typescript
// packages/core/src/atom-registry.ts

let deprecationWarned = false;

function warnDeprecated(fn: string): void {
  if (deprecationWarned) return;
  deprecationWarned = true;
  console.warn(
    `[nexus-state] ${fn} is deprecated. ` +
    `Every store is now self-contained. Use store.enableTimeTravel() ` +
    `and store.snapshot() instead. ` +
    `See MIGRATION.md for details.`
  );
}

export const atomRegistry = new Proxy(AtomRegistry.getInstance(), {
  get(target, prop) {
    warnDeprecated('atomRegistry');
    return Reflect.get(target, prop);
  }
});

/** @deprecated Use createStore() without arguments — every store is isolated by default */
export function createIsolatedRegistry(): AtomRegistry {
  warnDeprecated('createIsolatedRegistry');
  return new AtomRegistry();
}
```

### 2. Remove unused import from atom.ts

```diff
  // packages/core/src/atom.ts
- import { atomRegistry } from './atom-registry';

  // The import was there but never used in the function body.
  // Registration is deferred to first store access.
```

### 3. Update createStore signature

```typescript
// packages/core/src/store.ts

/**
 * Create a new store
 * @param plugins Array of plugins to apply to the store
 * @returns A new store instance
 *
 * @example
 * const store = createStore();
 * const storeWithPlugins = createStore([loggerPlugin, devtools()]);
 *
 * // SSR: each request creates an independent store
 * async function handleRequest() {
 *   const store = createStore(); // Already isolated!
 *   // ...
 * }
 */
export function createStore(plugins: Plugin[] = []): Store {
  return new StoreImpl(plugins);
}

/** @deprecated Pass registry directly — every store is already isolated */
export function createStoreWithRegistry(registry: AtomRegistry, plugins: Plugin[] = []): Store {
  warnDeprecated('createStoreWithRegistry');
  return new StoreImpl(plugins, registry);
}
```

```json
    }

```

## Phase 2: Removal (NEXT release — not this task)

After 1+ release cycle with deprecation warnings:

```diff
- export { atomRegistry, createIsolatedRegistry } from './atom-registry';
- export type { AtomRegistry } from './atom-registry';

+ // Removed: atomRegistry, createIsolatedRegistry
+ // Migration: Use store.snapshot() and store.enableTimeTravel() instead
```

Delete `atom-registry.ts` entirely.
