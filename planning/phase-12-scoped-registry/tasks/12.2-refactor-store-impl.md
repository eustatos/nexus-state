# Task 12.2: Refactor StoreImpl to use ScopedRegistry

| Meta | Value |
|------|-------|
| **Task** | Refactor `StoreImpl` to use `ScopedRegistry` instead of dual-registry |
| **Priority** | P0 — core refactoring |
| **Estimated effort** | Large |

| **Dependencies** | Task 12.1 |

## Context

`StoreImpl` currently coordinates 3 separate data structures. After Task 12.1, we have `ScopedRegistry` that unifies them. This task wires it in.
