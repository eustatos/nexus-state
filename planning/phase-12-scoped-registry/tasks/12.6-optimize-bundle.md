# Task 12.6: Optimize Bundle Size & Tree-Shaking

| Meta | Value |
|------|-------|
| **Task** | Verify and optimize bundle size after refactoring |
| **Priority** | P2 — validation + optimization |
| **Estimated effort** | Small |
| **Breaking changes** | No |
| **Dependencies** | Task 12.2, Task 12.3, Task 12.5 |

## Context

After all refactoring tasks, verify that tree-shaking works correctly and bundle size is reduced.

**Files to check:**
- `packages/core/dist/esm/` — ESM bundle
- `packages/core/package.json` — exports configuration
- `packages/core/.npmignore` — exclude test files

## Verification Steps

### 1. Build and measure

```bash
cd packages/core
npm run clean
npm run build

# Measure ESM only (production artifact)
powershell "(Get-ChildItem -Path dist/esm -Recurse -File -Exclude '*.map','*.d.ts*' | Measure-Object Length -Sum).Sum / 1KB"
```

### 2. Tree-shaking scenarios to test

| Import | Expected size | Notes |
|--------|--------------|-------|
| `import { atom, createStore }` | < 5 KB | Minimal: atom + store only |
| `import { atom, createStore, batch }` | < 6 KB | + batching |
| `import { atom, createStore } from '@nexus-state/core/debug'` | < 15 KB | + DevTools plugin |
| `import * from '@nexus-state/core'` | Full bundle | Expected |

### 3. Verify no dead code

```bash
# Check that atom-registry is NOT in minimal bundle
rollup --input _test-minimal.mjs --format esm | grep -c "atomRegistry"
# Should be 0

# Check that DevTools is NOT in minimal bundle
rollup --input _test-minimal.mjs --format esm | grep -c "REDUX_DEVTOOLS"
# Should be 0
```

### 4. Update .npmignore

```diff
  # Test files
  *.test.ts
+ dist/**/__tests__
+ dist/**/test-utils

  # Source files
  src/
```

### 5. Update package.json sideEffects

```json
{
  "sideEffects": false,
  "exports": {
    ".": {
      "types": "./dist/esm/index.d.ts",
      "import": "./dist/esm/index.js",
      "require": "./dist/cjs/index.js"
    },
    "./debug": {
      "types": "./dist/esm/debug.d.ts",
      "import": "./dist/esm/debug.js",
      "require": "./dist/cjs/debug.js"
    },
    "./utils": {
      "types": "./dist/esm/utils/index.d.ts",
      "import": "./dist/esm/utils/index.js",
      "require": "./dist/cjs/utils/index.js"
    },
    "./batching": {
      "types": "./dist/esm/batching.d.ts",
      "import": "./dist/esm/batching.js",
      "require": "./dist/cjs/batching.js"
    }
  }
}
```

## Acceptance Criteria

- [ ] ESM dist size (excl tests, maps, .d.ts) < 100 KB
- [ ] `atomRegistry` not in minimal bundle (atom + createStore only)
- [ ] `DevToolsPlugin` not in minimal bundle
- [ ] Test files excluded from published package
- [ ] All entry points work independently
- [ ] No circular dependencies
- [ ] TypeScript types generated for all entry points
